Postal3Script Language Documentation

I helped document an obscure scripting language used in the development of the worst games ever made, Postal III. If you want to mod Postal III in any meaningful capacity for whatever reason, then it's basically a requirement for you to learn this language, as it enables you to do tons of stuff!

You can read the documentation here: https://whackjobint.github.io/postal3script/
