Postal III Ultra Patch

The Postal III Ultra Patch is pretty much exactly what the title entails. This patch contains many fixes of core issues and crashes with Postal 3 that mostly I fixed while working on the Catharsis Reborn mod, but for Vanilla Postal 3 instead. These make the game much more playable. This patch also throws in some bonus cut features to play around with.
